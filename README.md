🌾 Work Exchange Platform for Rural Workers
📌 Overview

The Work Exchange Platform is a mobile application designed to connect rural workers with local employers. It enables workers to share their daily availability and allows employers to post job opportunities, improving communication and reducing unemployment in village areas.

This project focuses on creating a simple, accessible, and user-friendly solution for people with limited digital literacy.🎯 Problem Statement

In rural areas, workers face difficulty finding daily wage jobs due to the lack of a centralized system. Employers also struggle to find available workers quickly. Existing platforms are often complex and not suitable for low-literacy users.


💡 Solution

This application provides a digital platform where:

Workers can post their availability
Employers can post job requirements
Users can connect directly without intermediaries


🚀 Features
👷 For Workers
Register and login securely
Post daily work availability
Provide details like location, work type, and contact number
View available job opportunities

🏢 For Employers
Post job requirements
Specify work type, location, and daily wages
View available workers
Contact workers directly

🔥 General Features
Real-time data using Firebase Firestore
Secure authentication using Firebase
Simple and easy-to-use interface
Location-based filtering
